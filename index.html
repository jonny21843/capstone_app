About This Project

This static site is hosted on Amazon S3. You can upload categorized study notes into uploadedfiles/<field>/<category>/ and browse them here.

Fields represent domains like IT, Medical, Business, and so on. Inside each field, categories like Linux Notes or Anatomy Notes keep content organized.
